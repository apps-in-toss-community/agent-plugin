# claude-code-plugin

> 🚧 **Work in Progress** — not yet published.
> 아직 개발 중입니다. 릴리스 전입니다.

Official [Claude Code](https://claude.com/claude-code) plugin for building [Apps in Toss](https://toss.im/) mini-apps.

Claude Code 안에서 앱인토스 미니앱을 **생성·개발·테스트·배포**까지 한 번에 할 수 있게 해주는 공식 플러그인.

## Goal / 목표

`@ait-co/devtools`, `sdk-example`, `@ait-co/polyfill`, 공식 docs를 모두 엮어서 하나의 integrated Claude Code 경험을 제공:

- `/init` — 새 미니앱 스캐폴딩
- `/devtools` — devtools 패널 / mock 상태 인터랙티브 조작
- `/docs` — 큐레이트된 SDK 문서를 세션에 로드
- `/test` — sdk-example 기반 스모크 테스트 실행
- `/publish` — 앱인토스에 배포
